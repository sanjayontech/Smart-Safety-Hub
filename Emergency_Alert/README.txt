# Family Emergency Contact Alerting System

## Steps to Run

1. Install Python
2. Open terminal / command prompt
3. Install Twilio library:

pip install twilio

4. Open the Python file:
family_emergency_alert.py

5. Replace:
- YOUR_ACCOUNT_SID
- YOUR_AUTH_TOKEN
- Twilio number

with your Twilio account details.

6. Run the program:

python family_emergency_alert.py

7. Press E to trigger emergency alert.
